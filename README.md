# AutoBank - Fully Automatic Banking System

A modern, feature-rich banking web application built with vanilla HTML, CSS, and JavaScript.

## Features

- **User Authentication**: Register and login with account number and PIN
- **Dashboard**: View financial overview with balance, income, and expenses
- **Account Management**: Manage your bank accounts with visual credit card display
- **Money Transfer**: Transfer funds between accounts instantly
- **Transactions**: Complete transaction history with search and filtering
- **Auto Settings**: Configure automatic transfers and alerts
- **Responsive Design**: Works perfectly on desktop and mobile devices
- **Local Storage**: All data persists in browser local storage

## How to Use

1. **Extract the ZIP file** to your desired location
2. **Open `autobank.html`** in any modern web browser
3. **Create an account** by clicking the "Register" tab
   - Enter your full name
   - Choose account type (Saving or Current)
   - Set an initial deposit (minimum Rs. 500)
   - Create a 4-digit PIN
4. **Login** with your account number and PIN
5. **Explore the features**:
   - View dashboard with financial stats
   - Manage accounts and see credit cards
   - Transfer money to other accounts
   - Check transaction history
   - Configure auto-save rules

## Account Types

- **Saving Account** (Gold Card): For savings with interest
- **Current Account** (Platinum Card): For business transactions

## Technical Details

- **No Backend Required**: Fully client-side application
- **Local Storage**: Data persists across browser sessions
- **Modern Design**: Clean, professional UI with smooth animations
- **Responsive**: Optimized for all screen sizes
- **Security**: PIN-based authentication (demo only)
